# Flat Bullets

Status: active
Content type: TIPS

Bold category label + comma-separated items per line. Fast scanning for practical advice, gear reviews, rankings.

## Template

```
- **[Category]**: [item], [item], [item]
- **[Category]**: [item], [item]
```

## Example

```
- **Budget picks**: Samson Q2U ($70), Audio-Technica ATR2100x ($99)
- **Lighting**: Ring light for video calls, key light for streaming, avoid overhead fluorescents
- **Cable management**: Velcro straps over zip ties, label both ends, route behind desk
```

## Rules

- Bold category label followed by colon
- Items comma-separated on same line
- Keep items short — fragments, not sentences
- One category per bullet, group related items

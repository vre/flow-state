# Dialogue Essence

Status: archive
Content type: INTERVIEW (alternative)

Topic heading + speaker quotes + context bullets. Preserves who said what. More "editor's cut of the interview" than analytical summary. Useful when speaker dynamics matter (debates, disagreements).

## Template

```
### [Topic heading]
> **[Speaker]**: [compressed quote]
- [context/evidence]
> **[Other speaker]**: [response]
```

## Example

```
### The real advantage is parallelism, not speed
> **Boris**: 5-10 sessions daily. Half my coding on iOS. Kick off 3 from phone in the morning.
- Co-work handles spreadsheets and Slack-messages engineers autonomously
> **Greg**: How do you manage context across that many?
> **Boris**: You don't go deep — you tend to them like a generalist.
```

## Rules

- Heading: informative topic claim, not just a label
- Blockquotes for direct or compressed speaker quotes
- Context bullets between quotes for evidence and observations
- Identify speakers by name
- Use when speaker attribution and dialogue flow are essential to the content

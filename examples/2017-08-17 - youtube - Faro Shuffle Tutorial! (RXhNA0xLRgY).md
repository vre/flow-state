## Summary

**TL;DR**: Card weaving technique where each card interlaces—difficult but visually impressive.

**Prerequisites**: Standard deck, practice time

1. **Get grip**: Deck sideways in elevated mechanic's grip (thumb on top, 3 fingers back, index up top). Pinky out, riffle ~half deck with index pressure. Square edges with thumb/ring finger while pulling away.

2. **Position halves**: Left hand—thumb top-left, index on top, 2 fingers side. Right hand—thumb 3/4 down left edge, middle/ring fingers opposite, pinky supports bottom. Touch bottom corners of both packets.

3. **Weave cards**: Lightly touch corners, tilt/swoop angled. Sway right packet left-right while applying pressure. Cards interlace as you push together. Maintain steady thumb grip.

4. **Bridge finish**: Push packets together, loosen thumbs. Index underneath for support, back fingers apply pressure. Grab both ends with thumb/middle finger, hold tension. Release to bridge.

**Result**: Interlaced deck with visual flourish bridge.
